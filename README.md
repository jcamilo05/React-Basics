## Node and NPM

Before to start to create a React application is needed to have installed **Node** which is a Javascript run-time and it is designed to build scalable application; mostly things referred to backend part, and the other part that must be installed previously even before node is npm which is a package manager for node.js, **npm** will allow us to install all the libraries needed to run our React application.

To start the a react application without needed to setting up the initial configuration, write down the following line

` npm init react-app my-app`

The previous code will create a bunch of files, the most important ones are 

**Node_ modules**: here are placed all the node packages that have been installed via via npm

**Package.json**: This folder shows all node package json and other configurarion 

**Package-lock.json**:This file indicates npm how to break down all node package version
